Profile: Yuri Tricys

Little by little, a little becomes a lot.

 ‌

Hello, Dear Readers!

Hugo is the latest thing going for static website builders!

My name is Yuri. I'm a freelance web developer, and the current maintainer writer/web developer for this newsletter. Web development is a fast growing field. The engineers developing browser technologies, devices, web standards, and programming languages are a busy bunch. Every year there a lot of changes. These changes add up, so that mastering web development takes longer and longer. Meanwhile, websites depreciate. Business owners are a productive bunch, with savvy skills-sets that include leadership and learning on the fly. While some owners are happy to leave the details to the developers, others want to know everything about their website. It helps to be in control when something breaks. But there is just so much to learn about web development - and more and more every year - an owner can get distracted from their core business, and spend too much time learning web development. But there is another way! The mission of this newsletter is to make it easier for website owners to learn more about their websites, gradually. We aim to keep you updated about technology and industry changes that may apply to you. My own goal as a writer is to present relevant information in an easily understandable way, and to make reading this newsletter fun!
