Functions

What syntax is used to write a function statement?

^

First write the function reserved word ’function’ (lower case), follow it with a space and the function identifier (name of the function). Follow the identifier with a space and parenthesis containing any function parameters.

^

Write an opening curly brace on the same line, then a new line for the function body, followed by a closing curly brace on the next line.

Is a function statement called when it is declared?

^

No. A function statement must be called seperately from its declaration. However, function statements are hoisted and can be called in the script before they are declared.

What is the difference between a function argument and parameter?

^

The function parameter is a placeholder for variables that are passed into the function. The function arguments are the actual variables passed into the function when it is called.

How to examine the Math object in the console?

^

console.dir(Math);

What is the Math.random / Math.round combination?

^

Math.random() selects any number between 0 - 1,
Math.round() rounds its argument down to the next round integer.

How to pick any number between 1 and 100?

^

Math.floor(Math.random() * 100);

How to pick any number between 50 and 25?

^

Math.floor(Math.random() * (50 - 25 + 1)) + 25;

Write a function to pick a number between any two numbers

^

`function pickNumberBetween (min, max){
return Math.floor(Math.random() * (max - min + 1)) + min;
}`

What is a function expression?

^

function expressions are declared and assigned to a variable. The variable reserved words ’let’, ’const’, or ’var’ come first, then the ’=’ assignment operator, then the ’function’ reserved word.

^

Following this is the usual parameter parens and body block. The body block is closed out with a ’;’. Function expressions is not hoisted.

Is a function expression called when it is assigned?

^

Yes.

What is function scope?

^

Function scoped variable is valid anywhere in the scope of the function, even if it is declared in a block within in a function.

^

Use ’var’ to created function scoped variables. Later you will learn about the concept of ’closure’.

What is a method?

^

A method is a function declaration within an object. The function identifier is the key and the function declaration is the value (of key value pair in object).

^

To call the method, list the object followed by ’.’ syntax followed by key with (). example: object.method();

What is an anonymous function?

^

A function declared without an identifier

^

they can be:

  • passed in to other functions as arguments
  • used to protect scope
  • used as a callback argument
  • used in objects to declare methods

^

`(function (){console.log(“this is an anonymous function!”)})`

What is a callback function?

^

A callback function is passed as an argument into other JavaScript functions. Example the ’setTimeout()’ method (part of DOM api) takes a millisecond argument and a callback function argument.

^

When the timer expires, the function is called.
