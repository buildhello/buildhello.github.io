Spread Operator

What changes occurred to many data structures in ES6?

^

ES6 too many of the data structures from ES5 and made them iterable

What are iterables?

^

Iterables can be iterated through with iterators, which are functions that move from iteration point to iteration point and apply some callback function or operation to each value, either returning the changed value or mutating the iterable.

List some JavaScript iterables?

^

  • Strings
  • Arrays
  • Sets
  • Maps
  • Typed arrays

What is the spread operator?

^

The spread operator is an iterator. It can iterate through an iterable and return the results.

^

You need to enclose the spread operator in `{}` or `[]` then you write 3 period operators followed by the identifier of an iterable you want to spread

Give an example of iterating a string with the spread operator

^

let string = “HELLO”;
let spreadString = […string];
console.log(spreadString);

//returns the array [’H’, ’E’, ’L’,’L’,’O’];

Use the spread operator to spread a string into an object

^

let string = “HELLO”;
let spreadString = {…string};
console.log(spreadString);//returns {’0’:’H’, ’1’: ’E’, ’2’:’L’, ’3’:’L’, ’4’:’O’}

^

  • notice the keys are enclosed in strings, that is closer to JSON syntax than Object literal syntax

How to merge arrays by spreading them out together?

^

You just use multiple spread operators with array identifiers.
const letters1 = [’a’,’b’,’c’];
const letters2 = [’d’,’e’,’f’];
const letters3 = [’g’,’h’,’i’];

^

const letters = […letters1…letters2…letters3];
The order matters when spread into an array

Merge Arrays with Array.prototype.Concat()

^

const letters1 = [’a’,’b’,’c’];
const letters2 = [’d’,’e’,’f’];
const letters3 = [’g’,’h’,’i’];

^

letters1.concat(letters2.concat(letters3));

What is the difference between a shallow copy and a deep copy?

^

When a copy is ’shallow’ a change to the copy does not change the source.
A change to a deep copy, however, changes the source of the copy.

List some JavaScript copy operations that make shallow copies

^

  • Spread syntax
  • Array.prototype.concat()
  • Array.prototype.slice()

^

  • Array.from()
  • Object.assign()
  • Object.create()

Add an element to an array with Array.prototype.push() and the spread operator

^

players.push(’Salah’);
players = […players, ’Tielmans’];

Can the spread operator be used in function declaration and function execution?

^

Yes, the spread operator can be used in parameters when declaring a function and in arguments when calling a function.

What is the difference between the ’spread’ operator and the ’rest’ operator?

^

The ’spread’ operator is the ’rest’ operator. When you use the spread operator in the parameters of a function declaration, you can list it last and type ’…rest’ which represents any other arguments

Give an example of how to use the ’…rest’ operator

^

function letterFun (a, b, …rest){
let joined = [a,b, …rest];
console.log(joined);
}

^

letterFun(1,2,3,4); //console.log(joined) returns [1,2,3,4];
The spread operator collects all the arguments after the first two and passes them into the function
