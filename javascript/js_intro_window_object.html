What is an object?

^

An object is a complex data type that contains data structures. Objects have methods, which are functions that result in actions, and properties which values that form characeristics.

How to access a JavaScript object?

^

To access the methods or properties of an object, list the object followed by a period and the identifier of the method or ojbect. When called (run), a method identifier must include parentheses after the object.

^

sampleObject.sampleMethod() sampleObject.sampleProperty;

What are standard built-in objects?

^

Standard built-in objects are a subset of built-ins also called natives. Built-ins come with the JavaScript core.

^

  • primitives: undefined , null , Infinity , NaN
  • natives: Date() , Object() , String() , etc.
  • global functions: eval() , parseInt() , etc.
  • namespaces: Math , Atomics , JSON
  • friends of JS: Intl , WebAssembly

^

Natives are often called standard built-ins. They can be used to create new strings and numbers. They also have methods and properties you can work with.

How to access the representative string object?

^

Type into the console, String.prototype; Note the capitalized S

^

A variable containing a string can be called with any of the methods or properties on the string prototype. This works through protypal inheritance.

What is the globalThis object?

^

The globalThis object is the object that all other objects are a part of. They are created in the default namespece when the JavaScript environment runs.

What is globalThis in the broswer environment

^

In the browser, the globalThis object is called the window object

How to identify methods of the window object in the console?

^

Type window; into the browser console. The methods have an ’f’ next to the name, followed by ()

^

You don’t need to type window.alert();, because the window object is the globalThis object, so you can just type alert();

What is the document object model?

^

The document object model or DOM is the property of the window object web devlopers use the most. When the browser parses an HTML document into the DOM object, it creates an object for every HTML element called a node.

^

The nodes together are arranged in a hiierarchical tier-like structure called the DOM, along with other APIS functions and properties.

What is the difference between console.dir() and console.log()

^

console.log() logs arguments to the console. console.dir() shows the methods, nodes and properties of the arguument when the argument is a node.

How important is the document object model?

^

Almost all of the standard dynamic activity on a website is programmed with JavaScript using properties and methods to navigate and change the document object model.

What are some important methods to use with the DOM?

^

getElementByID(), querySelector(), querySelectorAll(), getElementsByClassName(), getElementsByName(), getElementsByTagName(), getElementsByTagNameNS()

What does document.getElementByID(); do?

^

document.getElementByID() crawls the DOM and returns the HTML node with the ID that matches the argement specified in the parentheses.

^

It can also be called from an element (rather than the document).

What does document.querySelector(); do?

^

document.querySelector() crawls the DOM and returns the first node that matches the argument passed into the querySelector, such as a class, an id, or other css selector.

^

It can also be called from an element (rather than the document).

What does document.querySelectorAll(); do?

^

document.querySelectorAll() crawls the DOM and returns a list of all the nodes that match the argument passed into the querySelector, such as a class, an id, or css selector.

^

It can also be called from an element (rather than the document).

What does document.getElementsByClassName(); do?

^

document.getElementsByClassName(); crawls the DOM and returns a list of all the nodes that match the argument passed into the document.getElementsByClassName();, such as a class, an id, or css selector.

^

It can also be called from an element (rather than the document).

What does document.getElementsName(); do?

^

document.getElementsName(); crawls the DOM and returns a list of all the nodes that match the name attribute argument passed into the document.getElementsName();

^

Remember ’name’ is an attribute typically attached to an <input> tag, but it can be used on other HTML elements.

^

document.getElementsName(); can also be called from an element (rather than the document).

What does document.getElementsByTagName(); do?

^

document.getElementsByTagName(); crawls the DOM and returns a list of all the nodes that match the tag name argument passed into the document.getElementsByTagName();

^

Remember tag name is the name of the HTML element. It could be <script> or <main> or <div>.

^

document.getElementsByTagName(); can also be called from an element (rather than the document).

What does document.getElementsByTagNameNS(); do?

^

document.getElementsByTagNameNS(); crawls a specified name space and returns a list of all the nodes that match the tag name argument passed into the document.getElementsByTagNameNS();

^

Remember tag name is the name of the HTML element. It could be <script> or <main> or <div>. Name space is a function, object, block or module.

^

document.getElementsByTagNameNS() can also be called from an element (rather than the document).
