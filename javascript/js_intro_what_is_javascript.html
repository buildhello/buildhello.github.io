What is JavaScript?

^

JavaScript is an interpreted, scripted, computer programming language. It’s written in a script, which is parsed at runtime, on the second pass, called compile time, it’s compiled into machine language. This process happens when you write any set of lines in the interpreter (REPL) and press enter.

Characteristics of JavaScript

^

  • platform independent
  • dynamically typed,
  • with functional style
  • object-oriented programming capability
  • prototypes
  • asynchrony

Who invented JavaScript

^

JavaScript was invented by Brendan Eich, for Netscape Communications, in 1955

What are JavaScript features?

^

Features are capabilities in the language that are added or changed each year once approved by the ECMA standard. Compilers, polyfills, browsers, servers, runtimes, and mobile devices add it by feature, so you need to check compatibility

How is JavaScript loaded?

^

JavaScript can be loaded in three ways:

  1. as an external script with <script></script>
  2. Directly in HTML in script tag
  3. Included directly in HTML elements: <button onclick=“jsFunction()”></button>
