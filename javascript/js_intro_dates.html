What can dates be used for in JavaScript?

^

Dates and times are used in web development for applications that need to display, measure, store, and/or modify time. Example: reminder application, scheduling application, clock, or current date.

Other date time formats

What is Unix Epoch time?

^

Unix Epoch time format is the number of milliseconds passed since the beginning of January 1st, 1970. Remember that date.

What is interational standard time format (ISO 8601)?

^

ISO 8601 is a year, month, day syntax, delimited with hyphens, followed by hours, minutes, seconds delimited by colons.

^

Example: “YYYY-MM-DD HH:MM:SS”

^

Sometimes ISO will include a ’T’ between the date and time blocks and/or a coordinated universal time (UTC) offset following the time block

Give an example of ISO time format?

^

“2019-02-07T23:28:34+00”, where the ’+00:00’ represents UTC offset of 0, or greenwhich median time.

^

Notice the use of the T, which makes it possible to write the time string in one string with no spaces. It’s not necessary to use the ’T’ in many cases

What is Unix Epoch time expressed in as a default?

^

Unix Epoch time is expressed in UTC (Coordinated Universal Time) timezones as a default

What is the standard built-in ’Date’ object in JavaScript?

^

The standard built-in ’Date’ object can be used to create new instances of a Date object.

^

It comes with methods and properties that can be used through prototypal inheritance to work with dates and times in JavaScript

How to create a new Date object instance?

^

To create a new Date object instance, use the new keyword with Date() and assign it to a variable.

^

Example:
let showTime = new Date();

The date object instance will represent the current time relative to January 1st 1970

What does a new Date Object instance represent?

^

A new Date object instance represents the current time since January 1st 1970. If Date() is called as a function, it will return a string representation of the current time.

^

However, browser or device may represent UTC time string returned with UTC offset.

What is the difference between creating a new date object instance and calling Date() as a function?

^

A new date object instance is an object that printed will show the current date and time is ISO format.

^

Calling Date() as a function, returns a string (not an object) with the current time as a string.

What kind of arguments does a new Date() object instance take?

^

A new Date() object instance takes all kinds of date arguments in strings, components, or milliseconds and coerces them into dates relative Epoch time (January 1st, 1970).

^

Dates are returned in ISO UTC, unless otherwise specified, with the offset appropriate to device

What happens if you pass 0 into a new Date object instance?

^

The new Date() object will return ’1970-01-01T00:00:00.000Z’, since Unix Epoch time in milliseconds is 0

String, Year, Month, Day

^

new Date(“January 1 1970”); returns 1970-01-01T08:00:00.000Z. 1st, 2nd, 3rd are not parsed by JavaScript Date objects

String, Day, Month, Year, Short Form

^

You can also pass months as abbreviations
new Date(“1 Jan 1970”);

Component 1970, 0, 1

^

You can also pass components into Date object instances
new Date(1970, 0, 1);

You can pass components delimited with hyphens “-” and components delimited with \\ commas ’,’

Are string components always parsed by a Date object instance?

^

new Date(’1970,0,1’) will not be parsed by a Date object, components separated by commas as a single string are not parsed.

^

new Date(’1970’, ’0’, ’1’); will be parsed by a date object.

String components delimited with commas are parsed by a date object.

Enter the time Neil Armstrong stepped on the moon into a date object as components (July 21st, 1969, 2:56 am, GMT)

^

new Date(1969, 6, 21, 2, 56); //returns 1969-07-20T09:56:00.000Z

^

Notice the 7th month, July, must be entered as the component 6, because JavaScript numbers start at 0.

Also notice, the time returned includes the UTC offset for your local computer.

What does Date.UTC() do?

^

Specifies the date components passed in, are in UTC with 0 offset (Greenwhich Meridian Time)

^

const tTime = new Date(Date.UTC(2023,10,10,15,41,30));

^

To get pacific standard add 7 hours to the time,
month argument comes second and starts with 0,
date argument comes third, and starts with 0.

How to return a human readable (web ready) date/time from a Date Object instance

^

To return a web ready date and time, use the Date.prototype.toLocaleDateString() method.

^

It takes a local argument as the first argument, and an options argument as the second argument
