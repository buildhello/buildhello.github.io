Arrays

Describe an array

^

An array is a special object used to hold data arranged in a list. Elements of the array (data items) are separated with commas, except for the last item in the list.

What are the positions of an array called?

^

The positions of an array are called ’indexes’, they are used to reference elements of the array. The leftmost position is always referenced using 0.

^

Each position rightward from the leftmost position increases sequentially by 1. You can think of array indexes as key value pairs, where the keys are hidden and fixed to numbers that match positions.

How to access an element of an array?

^

To access an element in an array, write the name of the array followed by square brackets containing the index referenced, or an expression resulting in the index referenced.

What does typeof() return when the argument is an array?

^

“const letters = [”a“, ”b“, ”c“];”
“console.log(typeof(letters)); //returns object”
An array is a special case of object.

How to change the value of a position in an array?

^

write an expression with the array identifier on the left-hand side followed by square brackets with the position number, then the = operator, then the value you want to assign to the array position on the right-hand side.

^

Close the statement with a semi-colon ’;’

How to return the last element of an array?

^

You an access the ’length’ property of an array through prototypal inheritance (we will talk about that later). Type the name of the array, then the period operator, the the word ’length’ with closing ’;’.

^

Subtract 1 to get the index for the last element in the array. You can return the last element, using the ’index’

Are arrays iterable?

^

Yes, arrays are iterable. You can iterate through them using methods that work with iterables, for example the ’for loop’

type an example of iterating through an array

^

for(let i = 0, i < array.length; i++){
console.log(array[i]);
}

^

The above will print all the elements of the array to the console

How to create an array in JavaScript?

^

In JavaScript you can create an array with direct assignment or by calling the Array constructor.

^

You can also create arrays as side-effects.

Create an array using direct assignment, and then with the Array constructor

^

To create an array with assignment, declare and assign or assign to a variable a square bracket with listed contents (or empty);

let aNewArray = [’1’];

Create an array with the array constructor

^

let newArray2 = new Array;

What is the difference between array creation with the array constructor and assignment

^

Both ’direct assignment’ and call the constructor create a new instance of the Array class, but the constructor can take a numerical argument that results in empty positions (test with length property)

Where do arrays get the methods and properties that can be used on them?

^

Arrays inherit methods and properties through prototypal inheritance from the global Array.prototype

What does Array.prototype.push() do?

^

Array.prototype.push(), adds a value to a new list item in the last position of the array.

^

Remember if you push an array onto another array, it adds the entire array into the last position, creating a nested array

What does Array.prototype.slice() do?

^

Array.prototype.slice() can be used to copy sections from an array. It takes two arguments. The first is the position to start slicing, the second is the position to stop slicing.

^

Negative arguments count from the right, and positive arguments count from the left. Array.prototype.slice() will return individual list items, if you want them in an array you must assign them to an array.

What does Array.prototype.concat() do?

^

Array.prototype.concat() joins two arrays together into the same array. Add the items in an array to another array by creating a new array and assigning the two arrays with concat:

^

let aNewArray = originalArray.concat(addedArray);

Can you use the concatenate operator ’+’ to merge arrays?

^

You can use the concatenate operator ’+’ to merge two arrays; however, it coerces their list items into strings before it merges them

^

(so if they are already strings, go ahead).

What does String() do?

^

String() turns the argument passed to it into a string. If you pass it an array, it will return every element of the array coerced to a string

What happens if you join two arrays with the concatenate operator ’+’?

^

If you join two arrays with the concatenate operator ’+’, it returns a string of all the items in the arrays separator by commas,

^

but it doesn’t insert a comma between the two arrays, it just smushes them together

Note about Array.prototype.flatMap() and Array.prototype.flat();

^

Array.prototype.flatMap() and Array.prototype.flat() were introduced in ECMAScript 2018 to simplify structures like nested calls to Array.prototype.concat();

What does Array.prototype.join() do?

^

Array.prototype.join() concatenates the elements of an array into a string, separated by the argument passed to the join method.

^

If you have an array of letters,
let letters = [’a’, ’b’, ’c’];
you can join it like this:
let newLetters = letters.join(’-’);

^

console.log(newLetters); will return the string, ’a-b-c’

What does Array.prototype.forEach() do?

^

Array.prototype.forEach() iterates through an array and applies a callback function to each element. It does not mutate the array it iterates through, the callback function mutates it.

^

Use an anonymous argument as the callback function and use another method, like Array.prototype.push() to assign the mutated results.

Give an example of using Array.prototype.push()

^

let letters = [’a’, ’b’, ’c’, ’d’];
let capLetters = [];
letters.forEach(function(letter){
capLetters.push(letter.toUpperCase());
}

^

console.log(capLetters);//returns [’A’,’B’,’C’,’D’]

What does Array.prototype.Map() do?

^

Array.prototype.Map() iterates through an array and applies a callback function to each element.

^

However, unlike Array.prototype.forEach(), it returns a new Array containing the mutated elements.

Use Array.prototype.Map() to mutate lowercase letters to uppercase and add to new array

^

let letters = [’a’,’b’,’c’];
const upperCase = letters.map(function(letter){
return letter.toUppercase();
)}

^

console.log(upperCase);//returns [’A’,’B’,’C’];

What does Array.prototype.filter() do?

^

Array.prototype.filter() iterates through an array and tests each element against some criteria.

^

If true, it returns the value into an array, if not it does not return the value (just moves to the next element and repeats the test)

Use Array.prototype.filter() move numbers from mixed array into a new array.

^

let mixedArray = [1, ’a’, 2, ’b’, 3, ’c’];
const numbers = mixedArray.filter(function(element){
if(Number.isInteger(element))
return true;
}

^

console.log(numbers);//returns [1,2,3]

filter falsy values out of an array with Array.prototype.filter()

^

const mixed = [’’,undefined, false, NaN, mull, ’all’, ’good’, ’here’];
const trueValues = mixed.filter(function(item){
if(item)
return true;
}

^

console.log(trueValues);//returns [’all’, ’good’, ’here’]

What does Array.prototype.indexOf() do?

^

Array.prototype.indexOf() iterates through an array and returns the index of the first element matching criteria specified in the first argument.

^

The second argument takes an index to start iterating from
(a - number in the second argument counts from the right)

Give an example of finding the index of the first NaN value in an array

^

const lotsOfData = [’a’, 1, undefined, ’b’,NaN, 2]
console.log(lotsOfData.indexOf(NaN));//returns 4,
console.log(lotsOfData.indexOf(NaN, 1);//returns 3, because it starts iterating at \\ the 1st position, not the 0 position
