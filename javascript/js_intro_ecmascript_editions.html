What is a JavaScript polyfill?

^

Polyfills are code snippets that ensure more modern code works on outdated browsers.
You include them directly in the code with conditional fences.

What are transpilers?

^

Transpilers also ensure modern code runs on outdated browers. They do this by transpiling code (compiling it) it compatible code.

^

Other transpilers languages into JavaScript, like babel and typescript.
