Arithmatic operators and expressions

What is a JavaScript expression?

^

A JS expression is a piece of code that returns a value when interpreted. In arithmetic expressions, numbers are relationally parsed according to operator instructions.

What is order of operations in JS

^

Division and multiplication occur before addition and subtraction. Order of operations also relies on associativity.

^

For example, left-associativity orders operators with the same precedence level from left-to-right, while right-associativity orders operators with the same precedence level from right-to-left

^

Assignment operators are right-associative

What are some operators?

^

`+` adds, `-` subtracts, `/` divides, `*` multiples, `**` raises to an exponent, `%` returns remainder of division

What is an assignment operator?

^

Assignment operators in JS declare, assign, and reassign values to variables. All arithmetic operators can be used as assignment operators.

What are the binary assignment operators?

^

’' is the equal assignment operator '+’ is the addition assignment operator ’-' is the subtraction assignment operator '+’ is the multiplication assignment operator

^

'/=' is division assignment operator '%= 'is remainder assignment operator '**=' is exponent assignment operator

what are left and right hand values

^

binary operators require two values to operator on, the left hand and right and sides.

What are unary operators?

^

Unary operators require only one variable to operate on. The unary plus operator `++` increments the variable by 1, the unary minus operator `–` decrements variable by 1

^

In fact, the 1 is invisible and can be any number, `5++` increments the variable by 5 and `5–` decrements the variable by 5, for example.

What is the ternary operator?

^

A shorthand form of a conditional if else statement. Condition is followed by `?`, if true statement follows condition and is followed by `:`, if false statement is always the last statement

^

`(condition) ? ifTrue : ifFalse;`

What are relational operators?

^

Relational operators are used to check literals or values against one another according to conditions. Conditional statements containing relational operators return boolean data types.

list some relational operators

^

`>` greater than `<` less than `<=` less than or equal

^

`>=` greater than or equal `==` loose equality `===` strict equality

What is the loose equality operator?

^

The double equal operator, `==`, is sometimes called the ’loose equality operator’. It uses coercion to convert both values to the same type before relationally comparing them.

^

The ’loose inequality operator’ `!=` does the same as the loose equality operator but converts the boolean value return to its opposite.

What is the strict equality operator?

^

The strict equality operator, `===`, checks equivalence and returns true only when both type and value are equal.

^

The strict inequality operator, `!==`, is the same as the strict equality operator, but converts the returned boolean value to its opposite.

What is the not-equal operator?

^

’!=’, is a loose equality operator that first converts different types to the same type before checking if contents are not equal.

^

It is formed with a combition of “Logical Not”, or ’!’ and the equal operator ’=’
