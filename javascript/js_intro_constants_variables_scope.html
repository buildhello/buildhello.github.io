What is namespace?

^

In programming, ’namespace’ refers to RAM reserved for variable and function assignments and definitions [Objects, Arrays, Classes, etc…] Office/Shipping

What is the global or default namespace

^

The default is the global namespace. Anything declared in global namespace is available in all other namespaces. In the console, it is called the ’window object’

What is variable scope?

^

Variable scope is the namespace in which a variable, function, or object is accessible

What are function, object, and block level scope?

^

Function level scope is only accessible with the function’s defining brackets. Block level scope is only accessible within the block’s defining brackets

What order is scoped accessed in?

^

Scope is accessed from outer to inner. Top level scope in any scope is accessed first. Hoisted variables are raised to top-level of their defining scope, during the first pass of the engine (runtime).

What are three ways to declare variables?

^

Using `let` (you can also use `const` or `var`)

  • let identifier1,var1,var3;
  • let var1 = “label”;
  • let var1 = var2 = var3 = “label

What is the difference between ’var’, ’let’ and ’const’?

^

  • ’Var’ is function level scope. It’s also hoisted to top level of its scope ,you can reference it before declaring.
  • ’Let’ has block level scope, you can’t reference it before declaring
  • ’Const’ has block level scope, it can’t be updated

Should variables be declared before they are assigned?

^

It’s good practice to declare all variables at top of scope.

Can you update properties of constants?

^

No, but if you assign an object to a constant, you can update the object’s properties.

Is var an outdated reserved word?

^

Var is function scoped. If you declare a variable in a block in a function and you want it available outside of the block but in the function use Var. Usually, however, ’let’ or ’const’ are more modern.

How to update a variable in JavaScript?

^

Assign a new value to a variable using the equal operator `=`, or binary operators `+=`, `-=`, `*=`, `/=`, `**=`, `%=`

Can one-side operators be used to update variables?

^

Yes, unary operators, like `++` and `–` can be used to increment (add 1) or decrement (remove 1) from a variable

Can you delete a variable in JavaScript?

^

No. You cannot delete a variable, but you can reassign it to the ’null’ value, which frees up the memory.

What is the ’delete’ reserved word used for?

^

The ’delete’ reserved word is used to delete properties of objects.
