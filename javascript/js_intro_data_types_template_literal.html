Data Types and the Template Literal

What are the JavaScript types?

^

JavaScript has primitive types and complex types.

What kind of data is used in Web Development?

^

Any data that can be accessed from a website, as well as: URLs, IP addresses, Email addresses, Analytics, theme data, headers, response codes, all of the data transfered between the client and server.

What are primitive data types?

^

Primitive data types consist of only one data type. They could be string, boolean, integer (number), floating point (number), undefined.

What is a complex data type?

^

A complex data can have two or more data types, examples are objects, arrays (special case of object), null.

What is coercion?

^

Coercion occurs when different primitive data types are used together in an operation. In such a case, one of the data types must be coerced to the other.

Are JavaScript numbers with decimals rounded?

^

Yes, JavaScript numbers with decimals (often called floating point numbers in programming) are rounded to 15 places. You specify how many places to round.

Are there specialty numbers in JavaScript?

^

Yes, `0` and `-0`, `Infinity`, `-Infinity`, and `NaN`
