Conditional Logic

What is a conditional statement

^

Computer programming is about decision logic. Progams respond to conditions and decide which code to follow.

^

Examples of conditional statements are: ’if’, ’if else’, ’if else if’ statements, ’while statements’, ’then’ statements (see async javascript)

How is an ’if’ statement written

^

In JavaScript, ’if’ statements are written with lowercase ’if’ followed by round parentheses.

^

The parentheses contain the conditions and the relational and logical operators used to check them.

What conditional statement always runs the code inside?

^

if(true) {} always runs the code inside.

Does a conditional statement require curly braces?

^

No, a conditional statement does not require curly braces.

Name some literals that are always false?

^

An empty string is always false. 0 and -0 are always false. ’NaN’, ’null’, and ’undefined’ are always false.

Name some literals that are always true

^

A string with contents is always true -Infinity and Infinity are always true

^

Positive or negative integers are always true. Empty curly braces or square braces are always true.

What are Logical And and Logical Not

^

’Logical And’ or && and ’Logical Not’ ! are used within conditional statements to check if multiple conditions return true or false

Is it recommended to use nested ’if else’ statements?

^

Code in general should be as minimal and easy to read as possible. Overly complex if else statements, including nested if else statements should be simplified.

What does Logical And (&&) do?

^

&& attempts to coerce lef and right side expressions to boolean values and returns the first false evaluation.

^

If both sides return true, && returns the last true evaluation. if (`!` && 587) returns true, both sides are true

What does Logical Or (||)?

^

Logical Or (||) attempts to coerce left and right side expressions to boolean values and returns the left side if it is true, and otherwise the right-side!

^

if(!`` || ``) returns ’true’ the left-side is true, an empty string is always false, the Logical Not plus an empty string is true, the right-side is not even tested

Are parentheses evaluated before everything else in conditional statements?

^

Yes, parentheses are evaluated before everything else in conditional statements.

if(-Infinity !== Infinity || NaN) what does return?

^

Returns True, Infinity is not the same as -Infinity, which is false, so Logical Not turns that false into a true, NaN is never evaluated.

Can you use equivelence operators with relational operators?

^

Relational operators, like < and = can be used with ||, &&, and ! in if statements. You can also use methods like typeof()

What is a common use case of a conditional statement

^

Conditional statements are often used to check values retrieved from data structures before they are fed into functions.

What should you watch out for when checking values in JavaScript?

^

Be careful with if(object.value) when checking for validity. 0 is often valid data, but it can break your code if you’re not careful.

How does the ’else’ reserved word work when following an ’if’ statement?

^

The ’else’ reserved word, immediately following an ’if’ statement behaves sort of like the || operator.

^

If the ’if’ statement returns true, the code after ’if’ runs, if it returns ’false’ the code after ’else’ runs.

Give an example of an ’if else’ statment

^

`if (conditional comparison){ //run some code } else if (conditional comparison){ // run some code } else { //run some code }`

What is a switch statement?

^

JavaScript switch statement makes it easier to write complex if else structures. The ’switch’ reserved word is followed by parentheses that contain an expression followed by square brackets. All the logic is between those brackets.

Describe switch statement syntax

^

’switch’ reserved word is followed by parentheses that contain an expression and curly braces that contain the statments logic.

^

Code runs base on values that match the expression’s outcome. The values are listed after the ’case’ reserved word followed by double colon : .

^

The last case uses the ’default:’ word. If no other case runs, then the default case runs. It can be left blank

Give an example of a switch statement

^

switch (varx >= 7){ case: true console.log(’The variable varx is greater than or equal to 7’); case: false console.log(’The variable varx is less than 7); default: console.log(’There is some problem with varx, so the condition wasn’t evaluated properly’); }//end switch statment

Does the switch statement automatically break out of the statement structure once a case that matches the evaluted expression is returned?

^

No. The switch statement always runs through all the code, unless a break statement is included, so always include break statements in the conditional code.

What is a ternary statement?

^

A ternary statement provides a terse ’if else’ structure.

Describe a ternary statement

^

The ’if’ condition is written without the word if in a parentheses followed by a question mark ? with some code after it, then a : with some code after it.

^

If the condition returns true the code after the ? runs, if it returns false, the code after : runs.

Give an example of a ternary statement

^

(varx > 7) ? console.log(`The variable ’varx’ is greater than 7`) : console.log(`The variable ’varx’ is not greater than 7`);
