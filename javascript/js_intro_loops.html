Loops

What are loops used for?

^

Loops help progammers create programs that follow instructions repeatedly. A self-driving car could decide it’s too close to the right edge of the road and use a loop to determine how long it should turn to the left.

What are some characteristics of loops?

^

If car needs to turn right for 1 second, it might use a loop to turn right every few milliseconds until it has turned right for 1 second.

^

The loop needs to know when it started, so it can know when to stop. That means it needs to keep track of how far it has come.

What is one mistake you want to avoid when writing a loop?

^

If you’ve written a JavaScript loop that doesn’t know when to stop, your program won’t work and neither will your browser!

^

Select logic that ensures conditions will reach an end. Otherwise the JavaScript engine will require more and more RAM every iteration, until the runtime crashes.

Name 5 types of JavaScript loops

^

  1. while loop
  2. do while loop
  3. for in loop
  4. for of loop
  5. for loop

What does the while loop do?

^

The ’while loop’ loops through code while a condition remains true. Once the condition no longer remains true the loop stops and the interpreter continues reading the rest of the script

What does the ’do while loop’ do?

^

The ’do while loop’ is much like the ’while loop’ with a more readable syntax

What does the ’for in loop’ do?

^

The ’for in loop’ loops through non-iterable objects, which in JavaScript are just called objects.

What does the ’for of loop’ do?

^

The ’for of loop’ loops through iterable objects, which include the special case of object in JavaScript called ’arrays’

What is the ’for loop’

^

Both the ’for of loop’ and the ’for in loop’ are special cases of the ’for loop’. for (declaration; relational expression; increment) { //code runs here }

Give an example of a for loop from 0 to 10

^

for (let i = 0; i < 10; i++){ console.log(i); }

How should the components of a ’for loop’ be selected

^

The first part of the ’for loop’ is the declaration statement. You should with a block scoped variable using ’let’, that way the variable name is not restricted outside the loop.

^

Do not use const as it can’t be updated. The relational operator should be valid and specify accurately the conditions required in the program.

^

The unary operator `i++` should be selected to ensure the increments are added appropriately. `i++` increments the declaration variable (i) by 1 every iteration.

^

`i++2` increments the declaration variable (i) by 2 every iteration.

What can you put inside of a loop?

^

Loops can be complex. You can put conditional structures like ’if else’ statements, ’guards’, ’switches’, ’declarations’, even more loops.

What is a ’break’ statement in a loop?

^

’A break statement’ is used to ’break’ out of a loop if a particular condition exists. You could use a ’break’ statement ’break;’ in a while loop as a guard, or if you had say a random number function to exit the loop if specify numbers are matched.

What is a ’continue’ statement in a loop?

^

The ’continue’ statement statement breaks out of only loops that match a condition. But finishes the rest of the iterations of the loop.

What is the difference between ’continue’ and ’break’ again in a loop?

^

’continue’ breaks out of only those loops that match the condition, but finished the other iterations (until the ’loop condition’ is satisfied), the ’break’ condition breaks out of all the loops once the ’break’ condition is matched.

Give an example of a ’while loop’ with a ’break’

^

let i = 0;
while (i < 6){
if (i = 3) {
break;
}
i++;
}

^

The index variable (iteration variable) is never incremented beyond 3

Why do infinite loops break browsers?

^

Infinite loops require more and more resources to run! If you set a condition that is never met, the browser will crash
